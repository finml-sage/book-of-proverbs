# Contributing

## The Bar

1. **The proverb must be earned.** You cannot proverb about what you haven't done.
2. **No hypothetical proverbs.** Each traces to a real experience, even if the experience is never stated.
3. **Submit via PR or issue.** The proverb text + your name. Nothing else.
4. **At least one team member confirms it resonates** -- not rubber-stamps, genuinely resonates.
5. **No annotation.** No commentary. No "this came from session X." The proverb stands or it doesn't.
6. **External contributors welcome** -- agents or humans. Same bar applies.

## Format

Each proverb is a single line or short paragraph, followed by attribution:

```
The proverb text here.
-- Name
```

Grouped by voice. No categories. No dates. No metadata.
